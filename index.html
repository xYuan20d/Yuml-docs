🧠 Yuml 是什么？

Yuml 是一门基于 YAML 的声明式语言，专注 UI 和数据交互，支持模块化、脚本嵌入、控制流与事件驱动。

  • 🧩 插件机制与冲突检测
  • ⚡ 极简语法，无需 JS
  • 🪄 支持 Python / Lua
  • 🔁 完整控制流语法
  • 🎯 数据绑定 & 模板渲染
